# 📈 Trade Graph IQ

**Trade Graph IQ** is a stock growth prediction web application that allows users to enter a stock symbol (e.g., `AAPL`, `GOOG`) and receive a visual prediction of its future performance. The project integrates a clean **HTML/CSS/JavaScript frontend** with a **Python Flask backend** for data processing, modeling, and charting.

---

## 🚀 Features

- 🔍 Input a stock symbol to get predictive insights
- 📊 Line graph showing past and predicted stock trends
- 🤖 Python-based prediction model (can be extended to use ML)
- 🧠 Built using Flask, Matplotlib, and financial data libraries
- 💻 Frontend built with HTML, CSS, and JavaScript
